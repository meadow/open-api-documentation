---
title: Permissions
---

The Meadow API is broken up into groups of endpoints that require a certain permission to access. An organization will be able to create/edit client api keys to control which permissions a key will have.

On the left sidebar, each group corresponds to one of the permissions detailed below.

# Analytics

Historical endpoints to retrieve data for an organization regarding their orders, products, purchase orders etc.

# Create Orders

Retrieve pricing for a cart and create orders for an organization

# Customers

View and modify information for an organization's customers

# Menu

Retrieve all menu items for use to display on a custom menu

# Products & Inventory

Create/edit information about products and their related entities
